O pluginie

Czym jest Granim.js ?

Granim.js jest to plugin JS stosowany do tworzenia gradientów i interaktywnych animacji . Dzięki Granim.js możemy stworzyć niestandardowe przejścia gradientowe w pełnym kolorze. Istnieją różne metody, których można użyć, aby gradient reagował na różne zdarzenia użytkownika, takie jak kliknięcie przycisku.

Czym jest Gradient ?

Jest to rozwiązanie stosowane w grafice komputerowej, polegające na wypełnieniu określonego obszaru płynnym przejściem pomiędzy co najmniej dwoma kolorami. Kształt tego wypełnienia może być rozmaity począwszy od prostego równomiernego Przejścia pomiędzy przeciwległymi bokami prostokąta a skończywszy na przejściu Wzdłuż linii krzywej, rozchodzeniu się promienistym itp.

Autorem pluginu jest: Benjamina Blonde

Case Study

Efekt krok po kroku

Krok 1: W HTML tworzymy blok canvas o id np. "canavas-image-blending"

Krok 2: Tworzymy styl do bloku "canavas-image-blending" ustawiamy szerokość, wysokość oraz określamy kolejność elementów wzdłuż osi z.

Krok 3: Tworzymy skrypt w js, który będzie odpowiadał za animacje

1.W sekcji element wybieramy blok dla którego ustawiamy gradient w naszym przypadku będzie to "#canavas-image-blending".

2.W sekcji direction wybieramy kierunek gradientu, jakie są do wyboru można przeczytać w dokumentacji my wybieramy "top-bottom".

W sekcji isPausedWhenNotInView określamy czy animacja ma się zatrzymywać gdy nie jesteśmy w widoku okna. My wybieramy "true"

4.Dodajemy zdjęcie dla którego chcemy tworzyć gradient, w sekcji source dodajemy lokalizacje zdjęcia a w sekcji blendingMide jak obraz ma się łączyć z gradientem my ustawiamy "multiply".

5.W sekcji gradents wybieramy kolory które mają się łączyć w gradient

6.W sekcji transitionSpeed określamy czas trwania animacji podczas zmiany stanu.

    
            $(function () {
                var granimInstance = new Granim({
                    element: '#canvas-image-blending',
                    direction: 'top-bottom',
                    isPausedWhenNotInView: true,
                    image : {
                        source: 'img/obraz.jpg',
                        blendingMode: 'multiply',
                    },
                    states : {
                        "default-state": {
                            gradients: [
                                ['#29323c', '#485563'],
                                ['#FF6B6B', '#556270'],
                                ['#80d3fe', '#7ea0c4'],
                                ['#f0ab51', '#eceba3']
                            ],
                            transitionSpeed: 7000
                        }
                    }
                });
            });

API v2.0.0

Własności Domyślnie Typ Opis
element / string or HTMLCanvasElement Selektor CSS dla canvas element lub HTMLCanvasElement sama (np „# granim-canvas” lub document.querySelector ('# granim-canvas')), który zostanie użyty do animacji gradientu.
nazwa false string To jest przedrostek używany dla nazwy klasy ciemności / jasności dodanej do options.elToSetClassOnelement w zależności od średniej jasności gradientu, klasa zostanie zaktualizowana podczas animacji. Jeśli nie ustawisz nazwy, klasa nie zostanie ustawiona.
elToSetClassOn body string Element do włączenia klasy ciemności / jasności (np „# canvas-wrapper”) Przydatne tylko wtedy, gdy ustawiszNazwa.
kierunek diagonal string Orientację gradientu, możesz wybrać między:
  • przekątna
  • lewo-prawo
  • góra-doł
  • promieniowy
customDirection / object Pozwala dostosować kierunek gradientu za pomocą pikseli lub wartości procentowych.Ta właściwość jest wymagana, gdy właściwośćkierunek jest ustawione nazwyczaj.
isPausedWhenNotInView false boolean Czy animacja zatrzymuje się, gdy nie jest w widoku okna? (Pomimo tego parametru animacja zawsze zatrzymuje się przy zmianie karty).
scrollDebounceThreshold 300 int Jaki jest próg odbicia przewijania (w ms)? Przydatne tylko wtedy, gdyisPausedWhenNotInView jest ustawione na prawdziwe.
stateTransitionSpeed 1000 int Czas trwania animacji przy zmianie stanu (w ms).
defaultStateName stan domyślny string Nazwa stanu domyślnego.
states / objekt Obiekt zawierający wszystkie stany, zobacz więcej informacji poniżej.
image / object Obiekt zawierający opcje obrazu, zobacz więcej informacji poniżej.

Opcje statyczne

Własności Domyślnie Rodzaj Opis
gradienty / Array of Arrays Możesz ustawić gradienty na dwa różne sposoby: 1. Prosty sposób , ustawiasz tylko kolory, które będą równomiernie rozłożone, np. 

                  [
          
                    ['rgba(255, 153, 102, .33)', '#ff5e62', ...],
                    ['hsla(144, 100%, 47%, .75)', 'hsl(210, 96%, 46%)', ...],
                    ...
                  ]
2. W skomplikowany sposób ustawiasz kolory i ich pozycje (w zakresie od 0 do 1), np 

                  [
                    [
                      { color: 'rgba(255, 153, 102, .33)', pos: .5 },
                      { color: '#ff5e62', pos: 1 }, ...
                    ], [
                      { color: 'hsla(144, 100%, 47%, .75)', pos: .1 },
                      { color: 'hsl(210, 96%, 46%)', pos: .6 }, ...
                    ],
                    ...
                  ]
Dopuszczalne kolory to: szesnastkowy, rgb, rgba, hsl i hsla. Wszystkie gradienty powinny zawierać tę samą liczbę kolorów.
TransitionSpeed 5000 int Czas przejścia między każdym gradientem (w ms).
loop true Boolean Kiedy animacja dotarła do ostatniego gradientu, czy się powtarza?

Opcje obrazka

Własności Domyślnie Rodzaj Opis
source / string Źródło Twojego obrazu, np „img / 800x200.jpg”.
position ["center","center"] array Pozycja obrazu w kanwie, reprezentowana jako plik [x, y] array. Możliwe wartości:
  • left
  • right
  • center
Możliwe wartości y:
  • top
  • center
  • bootom
stretchMode / string Czy obraz musi się rozciągać? Ta opcja jest przydatna w przypadku projektowania płynnego / responsywnego. Reprezentowany jako[x, y]szyk. Możliwe wartościx i y:
  • none
  • stretch
  • stretch-if-smaller
  • stretch-if-bigger

Callback

Własności Domyślnie Rodzaj Opis

onGradientChange false function Wyzwalane, gdy gradient zmienia się i zapętla.
onEnd false function Wywoływane po zakończeniu animacji.
